# Appendix A — Acronyms and Terms

| Term      | Meaning                                                     |
|-----------|-------------------------------------------------------------|
| ABI       | Application Binary Interface                                |
| ALFS      | Automated Linux From Scratch                                |
| BLFS      | Beyond Linux From Scratch                                   |
| BSP       | Board Support Package                                       |
| DTB       | Device Tree Blob                                            |
| DTC       | Device Tree Compiler                                        |
| EFI       | Extensible Firmware Interface                               |
| ELF       | Executable and Linkable Format                              |
| ESP       | EFI System Partition                                        |
| FHS       | Filesystem Hierarchy Standard                               |
| GRUB      | GRand Unified Bootloader                                    |
| LFS       | Linux From Scratch                                          |
| LPAR      | Logical Partition (IBM Z)                                   |
| MOK       | Machine Owner Key (Secure Boot)                             |
| OBP       | OpenBoot PROM (SPARC firmware)                              |
| OPAL      | Open Power Abstraction Layer (IBM POWER firmware)           |
| OF        | OpenFirmware (IEEE 1275)                                    |
| PLT       | Procedure Linkage Table                                     |
| SBI       | Supervisor Binary Interface (RISC-V)                        |
| SRM       | System Reference Manual console (Alpha firmware)            |
| UEFI      | Unified Extensible Firmware Interface                       |
