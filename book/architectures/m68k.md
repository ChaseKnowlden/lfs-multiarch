## Sub-architectures

The m68k port covers several distinct hardware platforms:

| Platform     | Boot method             | Notes                          |
|--------------|-------------------------|--------------------------------|
| Amiga        | `amiboot` (AmigaOS)     | or network/TFTP boot           |
| Atari        | `atari-bootstrap`       |                                |
| VME boards   | Motorola VMEbus systems | Varies by board                |
| Q40/Q60      | Q40boot                 |                                |
| Sun3         | netboot (PROM)          |                                |
| HP 300       | SYS_BOOT                |                                |
| NeXT         | NeXT bootloader         |                                |

## Platform Notes

- Use `--with-cpu=68020` as a safe minimum; adjust upward for your hardware
  (`68030`, `68040`, `68060`, `coldfire`).
- Cross-compilation is essentially the only practical approach for m68k
  given the age and slowness of the native hardware.
- The kernel `ARCH=m68k` build supports all sub-architectures; select the
  correct platform in `make menuconfig`.
