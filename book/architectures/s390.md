# s390 (IBM Z / s390x) Supplement

| Variable         | Value                           |
|------------------|---------------------------------|
| `LFS_TGT`        | `s390x-lfs-linux-gnu`           |
| `LFS_ARCH_LINUX` | `s390`                          |
| `LFS_BITS`       | 64                              |
| `LFS_ENDIAN`     | big                             |
| `LFS_ABI`        | default                         |
| `LFS_GCC_EXTRA`  | `--with-arch=z196`              |
| Bootloader       | zipl                            |
| Tier             | 3 — experimental                |

## Platform Notes

- Target hardware: IBM Z mainframes and LinuxONE, or an IBM Z LPAR/z/VM guest.
- For testing without real hardware, use Hercules emulator or the official
  IBM Z Community Cloud.
- `z196` (IBM zEnterprise 196, 2010) is the minimum recommended architecture
  level.  Adjust upward for newer hardware (`z13`, `z14`, `z15`, `z16`).
- Storage: DASD (Direct Access Storage Device) is the native disk type.
  Use `fdasd` to partition and `dasdfmt` to format DASD volumes.

## z/VM vs LPAR

| Environment | Notes                                      |
|-------------|--------------------------------------------|
| z/VM guest  | Most common for development/testing        |
| LPAR        | Native partition on real Z hardware        |
| KVM guest   | Supported on z/VM or LPAR with KVM enabled |
